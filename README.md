TT MMA – Enkel Webbplats

Detta är en enkel en-sides webbplats för TT MMA med platshållartexter som du kan redigera senare.

Struktur:
- index.html — innehåll och layout
- styles.css — färger, typografi och sektioner
- script.js — liten interaktivitet (mobilmeny, mjuk scroll)

Kör:
1) Öppna `index.html` i din webbläsare (dubbelklicka).
2) Uppdatera texter, tider och priser efter behov.
